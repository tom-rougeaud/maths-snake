# maths-snake
S'inspirant du légendaire jeu Snake, le joueur doit répondre à des questions portant sur les automatismes du lycée pro (équations, %, conversions, etc..).  Une grande liberté de configuration est laissée (chrono, difficulté, vitesse, notions abordées, nombre de vies..), permettant à chacun de jouer à la version de Maths Snake qui lui conviendra.
